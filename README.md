# WPA-1gx

A lightweight Progressive Web App demo with a carousel and a simple donate UI.

## Features

- Carousel on the main screen displaying photos.
- Donate button shows a bottom strip with donation options ($50, $100, $500).
- Basic PWA support via manifest and service worker.

## Run locally

```bash
python3 -m http.server 8000
```

Open: http://localhost:8000
